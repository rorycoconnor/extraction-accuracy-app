## Implementation Notes

### CSS Variables Approach
```css
/* All themes use same variable names */
.light {
  --primary: 221 83% 53%;  /* Blue */
}

.dark {
  --primary: 221 83% 53%;  /* Same blue */
}

.purple {
  --primary: 265 75% 65%;  /* Purple! */
}

/* Components use variables */

  background: hsl(var(--primary));

```

### No Hardcoded Colors
